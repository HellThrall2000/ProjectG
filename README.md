# Krishna AI Persona

This repository contains a Retrieval-Augmented Generation (RAG) system designed to emulate the persona of Krishna.

## Setup

1.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
2.  Set up your environment variables by creating a `.env` file and adding your `GROQ_API_KEY`.

## Running the application

```bash
python main.py
```
